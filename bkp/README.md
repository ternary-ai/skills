# Ternary Capital — Archive

Archived notebooks, datasets, and experiments from Ternary's research into cognitive financial agents.
Part of Ternary's open-source stack for agentic financial intelligence.

## What this is

A snapshot archive of earlier research work — notebooks, annotated datasets, and prototype implementations. Content here predates the current agent stack and is preserved for reference and reproducibility.

## Contents

- **Knowledge graphs** — entity extraction and relationship mapping notebooks
- **Named entity recognition** — BERT-based NER annotation with pre-training datasets
- **Synthetic data** — early SDK experiments for generating financial training data
- **SEC extraction** — 10-K document parsing and extraction workflows
- **Local LLM** — Llama 3.2 experiments for cost-effective inference

## Open Source

Part of [github.com/ternary-ai](https://github.com/ternary-ai) — open-source agents, models, synthetic data, prompt engineering resources, and scaffolding maintained for the community.
